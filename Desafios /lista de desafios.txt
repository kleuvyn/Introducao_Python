Semana 1: Criar uma Página de Apresentação:

    Desafio: Crie uma página HTML simples que inclua seu nome, uma breve descrição sobre você e uma imagem de perfil.
    Estilize a página usando CSS para torná-la visualmente atraente.
    Use Python para exibir a página localmente em um servidor simples.

Semana 2: Criar um Portfólio Online:

    Desafio: Crie uma página HTML que mostre seus projetos pessoais, incluindo imagens, descrições e links para cada projeto.
    Utilize CSS para estilizar o layout do portfólio e destacar os projetos.
    Use Python para adicionar funcionalidades dinâmicas, como carregar projetos de um arquivo de dados ou de um banco de dados.

Semana 3: Criar um Site de Receitas:

    Desafio: Crie uma página HTML que liste várias receitas de culinária, cada uma com seu próprio título, lista de ingredientes e instruções.
    Estilize a página usando CSS para torná-la fácil de navegar e atraente visualmente.
    Use Python para adicionar uma funcionalidade de busca que permita aos usuários encontrar receitas por nome ou ingrediente.

Semana 4: Criar um Calendário de Eventos:

    Desafio: Crie uma página HTML que exiba um calendário com eventos marcados para datas específicas.
    Estilize o calendário usando CSS para destacar os eventos e tornar a interface intuitiva.
    Use Python para adicionar funcionalidades como adicionar, editar e excluir eventos do calendário.

Semana 5: Criar um Blog de Viagens:

    Desafio: Crie uma página HTML que mostre suas experiências de viagem, incluindo fotos, descrições e dicas para cada destino.
    Utilize CSS para estilizar o blog e criar uma atmosfera envolvente para os leitores.
    Use Python para adicionar funcionalidades como comentários de usuários e uma seção de pesquisa por destino.

Semana 6: Criar um Site de Notícias:

    Use Python para adicionar funcionalidades como atualização automática de notícias e filtro por categoria.

Semana 7: Criar um Gerenciador de Finanças Pessoais:

    Desafio: Crie uma página HTML que permita aos usuários registrar suas despesas e receitas, com campos para valor, categoria e data.
    Utilize CSS para estilizar o gerenciador de finanças e facilitar a navegação e entrada de dados.
    Use Python para adicionar funcionalidades como calcular o saldo atual, gerar relatórios e exportar dados para um arquivo.

Semana 8: Criar um Site de Compras Online:

    Desafio: Crie uma página HTML que liste diferentes produtos para venda, cada um com sua própria imagem, descrição e preço.
    Estilize a página usando CSS para criar uma experiência de compra agradável e profissional.
    Use Python para adicionar funcionalidades como adicionar produtos ao carrinho, calcular o total da compra e processar pagamentos simulados.

Semana 9: Criar um Site de Avaliação de Filmes:

    Desafio: Crie uma página HTML que liste filmes populares, cada um com sua própria capa, título, classificação e sinopse.
